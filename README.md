
### Installation:
```bash
$ composer require constacloud/ebay-sdk
```

### Add below line to your controller
```php
use Ebaypackage\EbayAPI;
```

### Get Ebay Orders
```php
public function getOrders(){

    return $ebay->getEbayOrders($token,$id,$PageNumber,$OrdersPerPage);
}
```
